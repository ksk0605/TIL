# 정렬 알고리즘

## 버블 정렬 (Bubble Sort)
### 🔍 개념 설명
* 인접한 두 원소를 비교해서, 크기 순서에 맞지 않으면 서로 교환한다.
* 한 사이클 돌면 가장 큰 값이 맨 뒤로 간다.
* 최선의 경우 O(n), 최악의 경우 O(n^2), 평균 O(n^2)
* 거의 안씀, 더 좋은 알고리즘이 많음 
* [구현코드](https://github.com/ksk0605/TIL/blob/main/algorithm/sort/bubble.py)
